Pop Culture Hangman (2019)

Platform

This was a solo project. I programmed the game in python. It is a 1 Player game.

Description

This game of hangman is played with pop culture words for the four categories of Food, Music, Movies and Games.

Aspects of the game I am proud of?

I implemented an easy and hard mode that toggles between the answer being case sensitive or not. I also added a text based visual for the hangman.

If I were to redo this what would I change?

The case sensitive toggle can be a little weird so I'd go back and rework it. Another improvement could be the option for the player to input their own words allowing the game to be multiplayer.
